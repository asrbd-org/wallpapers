# 🖥️ ASRBD Wallpapers Repository

A curated collection of **official ASRBD desktop wallpapers** designed with a **futuristic cybersecurity aesthetic**, inspired by modern Linux wallpapers such as **Kali Linux’s premium yearly releases**.

This repository stores **year-wise wallpaper packs**, making it easy to maintain branding evolution over time.

## 🎨 Design Philosophy

Every wallpaper follows these principles:

* **Futuristic** → modern cyber defense aesthetics
* **Minimal Branding** → ASRBD logo only
* **Meaningful Storytelling** → threat intelligence, resilience, defense
* **Linux Premium Feel** → inspired by Kali-style elegance
* **Dark Mode Optimized** → ideal for developer and security workstations

---

## 🛠️ Creation Workflow

Wallpapers are concepted and generated using:

* **Google Whisk** for AI-assisted generation
* Manual post-processing for:

  * logo placement
  * contrast refinement
  * wallpaper scaling
  * monitor ratio optimization

Recommended export formats:

* 4K UHD
* 2560×1440
* 1920×1080
* Ultrawide 3440×1440

## 🤝 Contribution Guide

Want to contribute a yearly wallpaper pack?

1. Create a new year folder
2. Keep **ASRBD logo-only branding**
3. Maintain futuristic cyber-defense aesthetics
4. Add preview images in `assets/previews/`
5. Submit with concept meaning and prompt source
